# Airline Flight Performance Dashboard

## Project Objective
Analyze airline flight performance to identify trends, operational inefficiencies, and opportunities for resource optimization.

## Tool
Tableau

## Dashboard Requirements
- Size: 1200x800, maximum of 2 dashboards with navigation between them
- Visualizations should address the following questions:
  - Which airports and airlines are the worst performers in terms of cancellations and delays
  - Provide the ability to drill down into the reasons behind these issues
- KPIs: at least 3 quantitative metrics
- Timelines: show data trends over time
- Interactivity: at least one action, properly configured tooltips, filters, and parameters
- Dual axes: at least one visualization should use dual axes
- UX/UI: clean layout, adherence to Gestalt principles, proper use of containers
- Tableau Public publication: provide a link + downloaded .twbx version

## Additional
- Optional: additional visualizations, analytics, and enhancements
